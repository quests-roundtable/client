### Installation and Run

-> npm install
   ```sh
   npm install
   ```

-> Run the development server
   ```sh
   npm start
   ```
The client runs at [http://localhost:3000](http://localhost:3000).

-> Build the app for production to the `build` folder.\.
   ```sh
   npm run build
   ```
